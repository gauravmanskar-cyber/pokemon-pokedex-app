# Pokemon Pokedex App

A full stack Pokemon search application built using React, Spring Boot, and PokeAPI.

## Features

- Search Pokemon by name
- Display Pokemon image
- Display Pokemon abilities
- Display Pokemon stats
- Display Pokemon types
- Spring Boot REST API
- Caffeine Cache Implementation
- Responsive UI
- Error Handling
- Loading State

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Spring Boot
- RestTemplate
- Caffeine Cache

### API
- PokeAPI
